## 📚 Sobre o Projeto
Este arquivo HTML marca meus primeiros passos na **estilização de páginas web**, explorando diferentes formas de aplicar cores utilizando HTML e CSS inline. Aprendi sobre cores nomeadas, RGB, HSL e transparência!

## 📁 Arquivo
| Arquivo | Descrição |
|---------|-----------|
| `cores.html` | Demonstra as diferentes formas de aplicar cores em elementos HTML usando atributos e CSS inline |

## 🎯 O que aprendi

### Formas de Aplicar Cores:

#### 1️⃣ **Cores por Nome**
```html
<body bgcolor="Beige">
<p style="color:coral;">
<h2 style="background-color: brown; color: blue;">

## 🖥️ Exemplos do Código

<body bgcolor="Beige">
<p style="color:coral;">Definindo cores de texto</p>
<h2 style="color: hsl(50, 80%, 40%);">Testando cores em hsl</h2>

```
